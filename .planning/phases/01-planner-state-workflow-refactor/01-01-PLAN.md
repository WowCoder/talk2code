## Tasks

### Task 1: 重构 AgentState 定义

**Read first:** `backend/agents/state.py`

**Action:** 将现有 `AgentState` 改为：

```python
# 删除这些字段：
# - agent_outputs: Annotated[List[dict], operator.add]  # 不再需要 4 份自由文本输出

# 新增这些字段：
# - plan: Optional[dict]           # Planner 输出的结构化设计
# - validation_result: Optional[dict]  # Validator 检查结果 (pass/fail, issues, fix_instructions)
# - retry_count: int               # Coder 重试次数

# 保留这些字段：
# - requirement_id: int
# - requirement_content: str
# - current_step: str
# - code_files: Optional[List[dict]]
# - error: Optional[str]
# - dialogue_history: Annotated[List[dict], operator.add]
# - metadata: dict
```

最终 `AgentState` 包含：`requirement_id`, `requirement_content`, `plan`, `current_step`, `code_files`, `validation_result`, `retry_count`, `error`, `dialogue_history`, `metadata`。

**Acceptance criteria:**
- `state.py` 中无 `agent_outputs` 字段
- `state.py` 包含 `plan: Optional[dict]` 定义
- `state.py` 包含 `validation_result: Optional[dict]` 定义
- `state.py` 包含 `retry_count: int` 定义
- `python3 -c "from agents.state import AgentState; print('ok')"` 无报错
