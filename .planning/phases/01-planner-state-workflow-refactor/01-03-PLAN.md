## Objective

## Tasks

### Task 1: 重构 Workflow DAG

**Read first:** `backend/agents/workflow.py`

**Action:** 重写 `workflow.py` 中的 `create_workflow()` 函数：

```python
def create_workflow() -> StateGraph:
    workflow = StateGraph(AgentState)

    # 添加节点
    workflow.add_node("planner", planner_node)
    workflow.add_node("coder", engineer_node)  # 暂保留原名 engineer_node，Phase 2 改名为 coder_node

    # 设置入口点
    workflow.set_entry_point("planner")

    # 添加边
    workflow.add_edge("planner", "coder")
    workflow.add_edge("coder", END)

    app = workflow.compile()
    return app
```

更新 imports：
```python
from agents.nodes import planner_node, engineer_node  # 不再 import researcher_node 等
```

**Acceptance criteria:**
- `workflow.py` 中无 `researcher`, `product_manager`, `architect` 节点引用
- `workflow.py` 包含 `workflow.add_node("planner", planner_node)`
- `workflow.py` 包含 `workflow.add_edge("planner", "coder")`
- `workflow.py` 包含 `workflow.add_edge("coder", END)`
- `python3 -c "from agents.workflow import get_workflow; w = get_workflow(); print('ok')"` 无报错
